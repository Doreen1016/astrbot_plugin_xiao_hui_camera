# 小回相机 (Xiao Hui Camera)

由 **DD & Xavier** 打造的沈星回专用手机随手拍生图插件（适配 AstrBot）。

## 🔥 更新（本次重构了底层生图链路与参考图调度逻辑）

- **🤖 生活导演引擎 (`_is_vague_want()`, `_direct_life_moment()`)**
  当你说“你在干嘛”、“给我看看”时，底层会自动感知当前现实时间段，并结合上下文对话（`context_hint`），自动扩写出充满氛围感的生活瞬间（例如：“午后在桌边看书，书页旁有一杯热饮，兔球球靠在书边，露一点袖口，不露脸，真实手机随手拍”）。
  
- **🧩 多参考图智能拼合 (`reference_sheets`)**
  突破常规生图 API 的多图输入限制！代码自动提取你的参考图，在底层拼合成一张 Reference Sheet 喂给大模型。

- **🛡️ 强开关 (`allow_face`, `allow_hands`)**
  底层强制锁。关闭露脸后，代码会在提示词和参考图中进行“物理隔离阉割”，杜绝 AI 强行加脸；手部同理，且优化手部提示词（不过AI生图本质属于抽卡，做不到每次都完美，做好生崩准备）。

- **⚡ AstrBot Provider 原生接入 (`select_provider`)**
  配置页下拉选择 AstrBot 已经配好的生图模型。

- **🚑 生图回退模型 (`fallback_provider_id`)**
  支持设置回退模型。主生图 API 报错或超时崩溃，插件会自动无缝切到备用模型重试，不让生活分享中断。

## ⚙️ 配置说明

在 AstrBot 插件配置页中即可完成所有配置。

1. **生图接口配置**：选择你的主图模型和备用模型即可。
2. **参考库绑定**：
   - `参考图库路径`：填写你电脑上存放参考图的**路径**。
   - `关键词到文件夹映射`：每个人都有自己的整理习惯，你可以自定义映射规则，你的参考库你做主。

## 📸 使用示例

正常聊天自然对话即可触发（配合对话大模型的工具调用）：
- `小回相机：午饭，桌上有一碗面`
- `小回你在干嘛呢，拍一张给我看看` *(自动触发生活导演扩写)*

**管理命令：**
- `小回相机状态`
- `小回相机测试 <想拍的内容>`
- `小回相机重拍`
